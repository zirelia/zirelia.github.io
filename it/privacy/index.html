B2B Privacy Policy

Data di Validità: {{ site.time | date: "%d %B %Y" }}

1. Ambito della Policy

Questa Privacy Policy si applica a KineticMCP in qualità di fornitore software B2B. Distingue rigorosamente tra i visitatori del nostro sito web aziendale e i dati processati dalle vostre istanze self-hosted del software KineticMCP.

2. I Vostri Dati Cliente (Il Principio "Black Box")

Noi NON processiamo i vostri Dati Salesforce o Prompt AI.

KineticMCP è architettato come "Middleware Self-Hosted". Quando installate il nostro container sui vostri server (AWS, Azure, On-Premise):

  • I dati fluiscono strettamente tra Il Vostro Server <-> La Vostra Org Salesforce <-> Il Vostro Provider AI.
  • Non abbiamo alcuna capacità tecnica di intercettare, visualizzare o archiviare i vostri record CRM, prompt o logiche di business proprietarie.
  • Non siamo un "Data Processor" (Responsabile del Trattamento) per quanto riguarda i vostri dati aziendali interni ai sensi del GDPR/CCPA, poiché non vi accediamo mai.

3. Dati di Validazione Licenza

Per prevenire la pirateria e far rispettare i termini di licenza, il Software potrebbe contattare periodicamente i nostri Server di Licenza per verificarne la validità. Gli unici dati trasmessi durante questo "handshake" sono:

  • Hash della Chiave di Licenza.
  • IP/Hostname del Server (per il blocco della postazione).
  • Numero di Versione.

Nessun dato aziendale (record CRM) viene mai trasmesso in questo payload.

4. Sicurezza della Nostra Infrastruttura

Sebbene non ospitiamo le vostre istanze KineticMCP, manteniamo una sicurezza rigorosa sulla pipeline di distribuzione del codice (Docker Hub, Registri Privati) per garantire che gli artefatti software scaricati siano sicuri, firmati e privi di malware.

5. Contatti per la Compliance

Per i clienti Enterprise che richiedono Data Processing Agreements (DPA) relativi ai dati di fatturazione o gestione account, si prega di contattare direttamente il proprio rappresentante.
